Welcome to Map Creator!

Simple project for creating interactive maps using xlsx files with data

To create map with markers like this, you neeed have xlsx file with latitude, longitude and label for markers.

No matter how the columns are signed, when uploading the file, you just need to select the column names that correspond to the above mentioned fields.

Also, you need to choose whether the uniqueness of labels is important to you or not. The final touch will be the choice of icon and its colour.

You can download this map as html, or as image (Take snapshot). Also, if you logged in our app you can save it to dashboard.
